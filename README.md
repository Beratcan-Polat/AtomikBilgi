## 📖 Proje Hakkında

**AtomikBilgi**, periyodik tablodaki tüm 118 elementi detaylı bir şekilde tanıtan, tamamen Türkçe olarak hazırlanmış interaktif bir web sitesidir. Elementlerin fiziksel ve kimyasal özellikleri, kullanım alanları ve keşif hikayeleri gibi bilgilere kolayca ulaşabilirsiniz.


## ✨ Özellikler

- 🧪 **118 Element Detay Sayfası** — Her element için ayrı bir bilgi sayfası
- 🎨 **Renk Kodlu Periyodik Tablo** — Element grupları farklı renklerle gösterilir
- 👨‍🔬 **Kimya Öncüleri** — Mendeleev, Marie Curie, Dalton, Einstein ve daha fazlası
- 📋 **Element Grupları** — Alkali metaller, halojenler, soygazlar, lantanitler, aktinitler vb.
- 📝 **İlginç Listeler** — En pahalı elementler, en tehlikeli elementler, teknoloji elementleri ve daha fazlası
- 📱 **Responsive Tasarım** — Farklı ekran boyutlarına uyumlu modern arayüz
- 🧭 **Kolay Navigasyon** — Sidebar menü ile hızlı erişim

### Yerel Ortamda Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak isterseniz:

1. **Depoyu klonlayın:**
   ```bash
   git clone https://github.com/Beratcan-Polat/AtomikBilgi.git
   ```

2. **Proje klasörüne gidin:**
   ```bash
   cd AtomikBilgi
   ```

3. **`index.html` dosyasını tarayıcınızda açın** veya bir Live Server uzantısı kullanarak çalıştırın.

> 💡 **Not:** Proje tamamen statik HTML ve CSS ile yazılmıştır. Herhangi bir build aracı veya bağımlılık gerektirmez.

## 🛠️ Kullanılan Teknolojiler

- **HTML5** — Sayfa yapılandırması ve içerik
- **CSS3** — Stil, layout (CSS Grid), responsive tasarım
- **Font Awesome 6.4** — İkon kütüphanesi
- **CSS Grid Layout** — Periyodik tablo düzeni
